# Интеграция CRM WebStyle с MadeTask
Автоматизация выплат внешним подрядчикам через API MadeTask.

### Цель интеграции
Интеграция CRM WebStyle с MadeTask связывает CRM компании «WebStyle» с облачным сервисом MadeTask для:
- единого управления задачами внутренних и внешних исполнителей
- автоматических выплат дизайнерам после приёмки работы менеджером
- исключения ручных переводов бухгалтерией
